# marp-theme-academic

fork自用，修改了默认字体为Noto Sans CJK。

在makdown.marp.themes中添加该url：https://raw.githubusercontent.com/Flurry-L/marp-theme-academic/main/themes/academic.css

markdown文件头加入以下片段以使用：

```
---
marp: true
theme: academic
---
```
